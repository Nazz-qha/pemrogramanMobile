Form Registrasi Android - pekan ke-5

Aplikasi Android form registrasi lengkap sebagai tugas perkuliahan  pekan ke-5.

Fitur

- ✅ Complete Form dengan TextInputLayout (Nama, Email, Password, Confirm Password)
- ✅ Advanced Validation (real-time, format email, password match)
- ✅ Selection Controls (RadioGroup jenis kelamin + Checkbox hobi min 3)
- ✅ Spinner kota asal + AlertDialog konfirmasi submit
- ✅ Long Press gesture untuk aksi tambahan
- ✅ Reset form

Teknologi

- Android Studio
- Kotlin
- Material Design Components
- Min SDK: 24 (Android 7.0)
- Target SDK: 35 (Android 15)
